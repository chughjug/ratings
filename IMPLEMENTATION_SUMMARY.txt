================================================================================
  HOW TO USE
================================================================================

QUICK START (5 MINUTES):

1. Create Google Form at forms.google.com
   - Add questions: Player Name, Email, Rating, etc.
   
2. Get your Form ID from URL:
   https://forms.google.com/u/1/d/FORM_ID_HERE/edit

3. Open Tournament in Your System
   - Navigate to tournament
   - Click "Players" tab
   - Click green "Connect Google Form" button
   
4. Configure in Modal
   - Enter Form ID
   - Set API URL & Key
   - Choose import options
   - Click "Test Connection"
   - Click "Save Configuration"

5. Copy Generated Code
   - Click "Copy Configuration"
   - Paste into Google Apps Script

6. Setup Google Apps Script
   - Create Google Sheet at sheets.google.com
   - Extensions ‚Üí Apps Script
   - Paste generated config code
   - Paste rest of google-apps-script.js code
   - Run setupFormImport() function
   - Authorize when prompted

7. Test!
   - Fill out form
   - Wait ~1 minute
   - Player should appear in tournament
   - Check FormImportLog in Google Sheet
