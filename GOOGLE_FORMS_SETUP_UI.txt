STEP 1: GET YOUR FORM ID
================================================================================

1. Open your Google Form
2. Click the three-dot menu (‚ãÆ) ‚Üí "Get prefilled link" or go to the form URL
3. Your Form ID is in the URL: https://forms.google.com/u/1/d/FORM_ID/edit
4. Copy just the FORM_ID part (the long string)

Example Form URL:
https://forms.google.com/u/1/d/1A2b3C4d5E6f7G8h9I0j1K2l3M4n5O6p7Q8r9S0t/edit

Your Form ID:
1A2b3C4d5E6f7G8h9I0j1K2l3M4n5O6p7Q8r9S0t

================================================================================
STEP 2: CONFIGURE GOOGLE APPS SCRIPT
================================================================================

1. Open your Google Sheet
2. Go to Extensions ‚Üí Apps Script
3. Find the FORMS_CONFIG section (around line 410)
4. Update these values:

PASTE THIS INTO YOUR SCRIPT:
================================================================================

const FORMS_CONFIG = {
  ENABLE_FORM_IMPORT: true,                    // Set to true to enable
  FORM_ID: 'YOUR_FORM_ID_HERE',               // Paste your Form ID here
  API_BASE_URL: 'https://your-domain.com',    // Your tournament system URL
  API_KEY: 'demo-key-123',                     // Your API key
  TOURNAMENT_ID: 'your-tournament-id',         // Your tournament ID
  CHECK_INTERVAL: 5,                           // Check every 5 minutes
  RESPONSE_LIMIT: 100                          // Max 100 responses per import
};

================================================================================
STEP 3: SETUP THE AUTO-IMPORT
================================================================================

After updating the FORMS_CONFIG:

1. In the Apps Script editor, find the function: setupFormImport()
2. Click the Play button (‚ñ∂) to run it
3. Grant permission when prompted
4. You'll see a confirmation: "‚úÖ Form import configured!"

If it says "Form import is disabled":
‚Üí Make sure ENABLE_FORM_IMPORT is set to true

If it says "Please set your FORM_ID":
‚Üí Make sure FORM_ID is filled in (not 'your-form-id-here')

================================================================================
STEP 4: WHAT HAPPENS NEXT
================================================================================

‚úÖ Real-time Import:
   When someone submits your form, they're automatically imported to the tournament

‚úÖ Backup Periodic Check:
   Every 5 minutes, the script checks for any missed responses

‚úÖ Auto-lookup:
   Player ratings and sections are automatically looked up and assigned

‚úÖ Confirmation Emails:
   Players receive a registration confirmation (if email field is mapped)

STEP 7: TROUBLESHOOTING
================================================================================
